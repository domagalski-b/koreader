📚 KOReader Custom Stuff (SimpleUI Tweaks & More)

A small collection of tweaks, experiments and “why is this not already like this?” improvements for KOReader — mainly focused on SimpleUI.

This repo started as a bunch of random patches and slowly turned into something slightly more organized (don’t worry, only slightly).

⸻

🧩 What’s inside

🔌 plugin/

The main attraction.

A custom plugin that:

* unifies multiple SimpleUI visual tweaks
* replaces fragile patches with a more stable approach
* makes the homescreen actually look like you want it to

👉 Full details + proper README live inside this folder

⸻

🖼 backgrounds/

Just wallpapers.

* Designed specifically to work with the plugin
* Made to align with UI layout (so things don’t look… off)
* Drop in, test, tweak, repeat

⸻

🧪 patches/

The experimental playground.

* standalone tweaks
* random ideas (some useful, some questionable)

Includes things like:

* library view tweaks
* font adjustments
* other small UI changes

👉 Use at your own risk (or curiosity)

⸻

🤔 Why this exists

Because:

* SimpleUI is great… but not quite there
* patches are powerful… but fragile
* and sometimes you just want your Kobo to not look like a default device from 2012

So this repo is:
👉 part utility
👉 part experimentation
👉 part “let’s see if this breaks everything”

⸻

⚠️ Important

* My setup is mix of the plugin and those patches applied
* Things may break after KOReader updates (that’s part of the fun)
* Always keep backups before testing

⸻

🚀 Status

Actively used, constantly tweaked, occasionally broken and then fixed again.

⸻
